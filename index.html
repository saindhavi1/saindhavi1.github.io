Dual Axis Solar Panel

This solar panel tracker was built using Arduino Uno and tested my skills in not only software but hardware components as I was required to create an execute a circuit design. This solar panel tracker will move in the direction of the light using LDR sensors to detect the light and servo motors to move the solar panel. This project won first place for the Kids for Kids Toronto Coding Club's first ever hackathon, GenZCanHack! Look in the README file to see our presentation and a video of how the solar panel works!


Language: Arduino C++

CPU Simulator

This application simulates the inner workings of the ARM v7 16 bit Central Processing Unit (CPU) of a computer, specifically in terms of storing values in registries and perform arithmetic operations. This CPU simulator allows the user to type in certain commands within assembly language to perform certain arithmetic operations and then show the CPU in action! This simulator also has a special register called the CPSR (Current Program Status Register). This register will tell if the answer to an operationis an Integer (I), Negative number (N), Zero (Z), has a Carry (C) or overflows (V) by showing their respective flags in the brackets.


Language: Java

Find My University Map Application

Collaboratively created an application for highschoo seniors looking for information about universities. I was tasked to create a map of all the universities in southern Ontario using the JXMapViewer API. The map is interactive as will filter by distance, the saved programs of the user and what universities you want to select. You will also be able to click on the university's logo to find more information about the university!
